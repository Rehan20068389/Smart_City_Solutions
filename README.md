Description:
Urban Solutions is a booking platform that allows users to book cars and cooks. It provides a smooth,
user-friendly interface for customers while also offering an admin dashboard for managing users, bookings, 
and service providers. The system is built using the MERN stack with a MySQL database.

Key Features:

Users can book cars or hire cooks based on availability.

Admin panel to manage users, bookings, and service providers.

Authentication system using JWT for secure login and session management.

Responsive frontend using React and Material-UI.

Backend API using Node.js, Express, and Sequelize ORM with MySQL.

Deployed: Frontend on Netlify, backend on Railway or Render.

How It Works

Users register or log in to the platform.

They can view available cars or cooks and make bookings.

Admins have a dashboard to:

View all users and service providers.

Approve, edit, or cancel bookings.

Manage available cars and cooks.

JWT Authentication ensures secure sessions for users and admins.

Axios communicates with the backend API to fetch or update booking data.
