 Urban Solutions Booking System

A MERN stack application for booking cars and cooks, with admin management, built with React, Node.js, Express, and MySQL. The frontend is deployed on Netlify and backend on Render.

---

## 游댢 Technologies Used

- **Frontend:** React, React Router, Material-UI, Axios
- **Backend:** Node.js, Express, MySQL (Sequelize ORM)
- **Deployment:** Frontend on Netlify, Backend on Render
- **Authentication:** JWT
